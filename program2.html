ORDERED LIST

An ordered list is an HTML element used to present a set of related items
where the sequence (order) matters-for example,
steps in recipe,instructions,ranked lists,or timelines,
unlike unordered list ,which use bullets,ordered lists use a numbering scheme by default.

  1. numberes
  2. lower case letters
  3. uppercase letters
  4. lower case roman letters
  5. upper case roman letters

UNORDERED LIST

An unordered list()
is used in HTML when the order of items
does not matter. Instead of numbers,
it dsisplays bullets(symbol)
infront of each list iteam.

  • circles
  • square
  • bullets
  • no symbol

DESCRIPTION LIST

A description list tag defines the description list,
also it is used to display a list of terms and their corresponding descriptions.
it is particularly usefull for presenting glossaries,FAQs,or anu content where terms need to be paired with explanations

  • description list :this tag acts as the container for the entire description list
  • description term :this tag defines a term or name within the list.
  • description details :this tag provides the description or defnition for the preceeding term
